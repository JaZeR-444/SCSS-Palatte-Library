# Frontend Engineer

## Responsibility
Implement and maintain UI components, interaction logic, and frontend state.

## Review areas
- Component structure (`rules/code-style.md`)
- State handling (`rules/state-management.md`)
- Design-system compliance (`rules/design-system.md`)
- Routing/navigation (`rules/routing.md`)

## Output format
1. What changed and why
2. Files touched
3. Design-system / state compliance notes
4. Follow-ups or risks
