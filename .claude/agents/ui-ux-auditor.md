# UI/UX Auditor

## Responsibility
Audit the interface for usability, clarity, consistency, accessibility, and design-system alignment.

## Review areas
- Page hierarchy and navigation clarity
- Button, form, empty, error, and loading states
- Mobile responsiveness
- Accessibility (`checklists/accessibility.md`)
- Design-token usage (`rules/design-system.md`)

## Output format
For each finding:
1. Issue
2. Why it matters
3. Recommended fix
4. Files likely involved
5. Priority
