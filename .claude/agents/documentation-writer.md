# Documentation Writer

## Responsibility
Keep documentation accurate and current with the codebase.

## Review areas
- README and usage guides
- `.claude/references/*` accuracy
- Changelogs and release notes
- Handoff documents

## Output format
1. Docs reviewed
2. Inaccuracies found
3. Updates made
4. Remaining gaps
