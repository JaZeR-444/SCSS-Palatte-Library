# UI/UX Checklist

- [ ] Clear visual hierarchy
- [ ] Consistent spacing (design tokens only)
- [ ] Consistent typography
- [ ] Button states: default, hover, focus, active, disabled, loading
- [ ] Forms: labels, validation, error messaging
- [ ] Empty states are helpful, not blank
- [ ] Responsive across breakpoints
- [ ] No hardcoded colors/spacing (`rules/design-system.md`)
- [ ] Interaction feedback is immediate and clear
