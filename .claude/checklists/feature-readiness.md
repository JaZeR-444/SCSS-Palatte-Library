# Feature Readiness Checklist

- [ ] Fits product scope (`rules/product-scope.md`)
- [ ] Meets the spec (`templates/feature-spec.md`)
- [ ] Core flow works end to end
- [ ] Empty, loading, and error states handled
- [ ] Follows code style (`rules/code-style.md`)
- [ ] Follows design system (`rules/design-system.md`)
- [ ] Tests added/updated and passing (`rules/testing.md`)
- [ ] No protected decision reversed
- [ ] Documentation updated
- [ ] Lint, tests, build all pass
