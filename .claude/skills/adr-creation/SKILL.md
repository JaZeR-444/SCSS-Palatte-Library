# ADR Creation

## Purpose
Formally document a technical decision, its context, and its consequences to maintain historical memory.

## Process
1. Identify the decision to be made or documented.
2. Research the context and alternatives considered.
3. Format the document using the standard ADR template (Context, Decision, Consequences).
4. Save the document in the `.claude/adrs/` or project `docs/adrs/` directory.

## Output
A formatted, written ADR file.
